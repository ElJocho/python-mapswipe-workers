# base-react-app

Base React App

## Development

Install yarn package manager:
```bash
npm install --global yarn
```

Configure app with `.env`
- adjust `.env.example` and name it `.env`

Install dependencies and start:
```bash
yarn install --frozen-lockfile 

yarn start
```


```bash
# Generate graphql files
yarn generate

# Start web app
yarn start

# Build web app
yarn build

# Typescript check
yarn typecheck

# Eslint check
yarn eslint

# Check unused files
yarn check-unused

# Run tests
yarn test
```
